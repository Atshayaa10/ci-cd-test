# Calculator
================

A simple calculator that calculates the average of a list of numbers.

## Features

* Calculate the average of a list of numbers
* Handle empty lists and single-element lists

## Usage

1. Install the package using pip: `pip install calculator`
2. Import the `calculate_average` function in your Python script: `from calculator import calculate_average`
3. Call the `calculate_average` function with a list of numbers: `result = calculate_average([10, 20, 30])`
4. Print the result: `print(f"The average is: {result}")`
